Raspberry Pi Setup

These instructions create a headless raspberry pi for your robot

  • Download Raspbian Buster Lite from https://www.raspberrypi.org/downloads/raspbian/

  • Extract the Raspbian .img from the .zip file

  • Download Rufus to burn the SD card https://rufus.ie/

  • Launch Rufus and select the .img file and click Start to burn the SD card

  • After rufus is finished you will see a new drive on your computer named "boot"

  • Right click select New -> Text File. Name the file "wpa_supplicant.conf"

  • Use Notepad to edit the "wpa_supplicant.conf" file you just created.

  • Paste this into the file and update the wifi login info

    • ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev
update_config=1
country=US
network={
    ssid="WifiNameHere"
    psk="WifiPasswordHere"
}

  • Right click select New -> Text File. Name the file "ssh". Leave this file blank. This will enable SSH access.

  • Remove your SD card and install into the raspberry pi.

  • Plug in the power to boot the raspberry pi.

  • Download and install Putty

  • Launch putty and enter "raspberrypi.local" for Host Name. Click Open.

  • If your pi is successfully connecting to your wifi you will get a "login as:" prompt. Type "pi" for the user. The password is "raspberry"

  • After putty is connected run the commands to install and run RobotDriver

  • Install RobotDriver: curl -L https://install.robotdriver.dev | sudo bash

  • Run RobotDriver: sudo node /usr/local/robotdriver/robotdriver.js

  • After RobotDriver is running open your web browser to http://raspberrypi.local
